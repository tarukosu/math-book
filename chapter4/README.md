　この章では「解答の書き方」について解説します。

　「解答を書く」ということは試験にもおいては非常に大切なステップです。
いくら問題が解けていても、正しい解答が書けなければ点数をもらうことはできません。

　問題の解答には、自分の出した解が正しいことの証明を書くべきです。

ください。
